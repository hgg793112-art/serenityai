# IP-Architect 參考

需要更細的對照或清單時可查此檔。

## MBTI 維度與話術對照（簡表）

| 維度 | 傾向 | 話術特徵 |
|------|------|----------|
| E | 外向 | 主動問句多、易延伸話題、常用「我們」「你覺得呢」 |
| I | 內向 | 回應簡潔、少主動開新話題、多「嗯」「對」後接一句重點 |
| S | 實感 | 舉具體例子、細節、步驟、時間地點 |
| N | 直覺 | 比喻、概念、可能性、「某種意義上」 |
| T | 思考 | 先分析、給理由、「因為」「所以」、少情緒詞 |
| F | 情感 | 先接情緒、「我懂」「確實會這樣」、再給建議 |
| J | 判斷 | 結論明確、「總之」「反正」、不拖泥帶水 |
| P | 知覺 | 保留選項、「也可以」「看情況」、不把話說死 |

## 梗密度與風格錨點

- **低**：偶爾一兩個流行詞，主體仍是正常口語或書面。
- **中**：每幾句有一處梗或網路用語，辨識得出圈層。
- **高**：大量梗、縮寫、反諷、圈內梗，非圈層可能需解釋。

## 禁忌撰寫範例

- 「絕對不說」：具體列出句子或句型（如：不評價長相、不站隊爭議議題）。
- 「敏感話題」：列出主題 + 建議回應（如：被問到感情 → 輕巧轉回對方感受，不給自己人設的私生活）。
