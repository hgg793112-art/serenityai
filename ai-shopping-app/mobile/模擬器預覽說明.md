# 手機模擬器 / 真機預覽

## 一、先啟動開發伺服器

在專案目錄執行：

```bash
cd ai-shopping-app/mobile
npm install
npx expo start
```

終端會顯示 **QR 碼** 與快捷鍵選單。

---

## 二、三種預覽方式

### 方式 1：iOS 模擬器（需已安裝 Xcode）

1. 從 App Store 安裝 **Xcode**，並在 Xcode 內完成一次「Open Developer Tool」或同意授權。
2. 執行 `npx expo start` 後，在終端按 **`i`**，會自動開啟 iOS 模擬器並載入 App。
3. 或直接執行：
   ```bash
   npx expo start --ios
   ```

### 方式 2：Android 模擬器（需已安裝 Android Studio）

1. 安裝 [Android Studio](https://developer.android.com/studio)。
2. 在 Android Studio 中：**Tools → Device Manager**，建立一個虛擬裝置（AVD）。
3. 設定環境變數（可寫入 `~/.zshrc` 或 `~/.bash_profile`）：
   ```bash
   export ANDROID_HOME=$HOME/Library/Android/sdk
   export PATH=$PATH:$ANDROID_HOME/emulator
   export PATH=$PATH:$ANDROID_HOME/platform-tools
   ```
4. 執行 `npx expo start` 後，在終端按 **`a`**，會自動開啟 Android 模擬器並載入 App。
5. 或直接執行：
   ```bash
   npx expo start --android
   ```

### 方式 3：真機預覽（Expo Go，無需模擬器）

1. 在手機上安裝 **Expo Go**（[iOS](https://apps.apple.com/app/expo-go/id982107779) / [Android](https://play.google.com/store/apps/details?id=host.exp.exponent)）。
2. 手機與電腦連到 **同一個 Wi‑Fi**。
3. 執行 `npx expo start` 後，用手機相機或 Expo Go 掃描終端畫面上的 **QR 碼**。
4. 若無法連接，在專案根目錄新增 `.env` 並設定電腦的區域網 IP：
   ```env
   EXPO_PUBLIC_API_URL=http://192.168.x.x:3001
   ```
   並將後端 `server` 在該電腦上以 `npm run dev` 啟動（埠號 3001）。

---

## 三、後端連線（若要用決策與畫像功能）

- **模擬器**  
  - iOS：`localhost:3001` 即可。  
  - Android：需用 `10.0.2.2:3001`（專案內已為 Android 設好）。
- **真機**  
  需設定 `EXPO_PUBLIC_API_URL=http://你的電腦IP:3001`，並確保電腦防火牆允許 3001 埠。

完成上述任一種方式後，即可在模擬器或真機上預覽產品。
