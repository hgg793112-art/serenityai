## 如何驗證

1. **瀏覽器打開**：http://localhost:3000  
2. **語音**：底部「壓力」→ 點開始 → 允許麥克風 → 應出現「我正在聽」並可對話。  
3. **文字對話**：首頁「小寧聊天」→ 「和小寧聊聊（有記憶的對話）」→ 輸入訊息送出 → 應收到小寧回覆；多輪對話會保留近期記錄。

若 dev 已關閉，在專案根目錄執行：

```bash
npm run dev
```

再打開 http://localhost:3000 即可。

## 模擬器／預覽測試步驟

1. **啟動**：在專案根目錄執行 `npm run dev`，瀏覽器打開 http://localhost:3000（或用手機/模擬器連同一網段的 `http://<本機IP>:3000`）。
2. **語音**：底部點「壓力」→ 點「開始」→ 允許麥克風 → 應出現「我正在聽」並可與小寧語音對話；結束後點「結束」。
3. **文字對話**：首頁點「和小寧聊聊」→ 點「和小寧聊聊（有記憶的對話）」→ 輸入一句話點「送出」→ 應收到小寧回覆；再發幾句可驗證近期記憶。
4. **錯誤時**：若語音顯示「請檢查 .env.local 的 VITE_GEMINI_API_KEY」或聊天顯示「請在 .env.local 設定…」，表示 Key 未載入，請確認 `.env.local` 已設定並重啟 `npm run dev`。
