## 音頻品質

- 所有 MP3 檔案：128-256 kbps, 44.1 kHz, 立體聲
- WAV 檔案：無損品質
- 所有音頻均支援循環播放

## 測試工具

訪問 `/audio/test-audio.html` 可測試所有音頻檔案的播放狀態。

## 授權說明

- **SoundBible 音頻**：需要在應用中註明來源（Attribution 3.0）
- **Public Domain**：無需註明來源
- **本地音頻**：專案內部使用

## 使用建議

1. 所有音頻已設定為循環播放（`audio.loop = true`）
2. 預設音量為 60%，可透過 UI 調整
3. 音頻載入時會自動淡入，停止時會淡出
4. 如果音頻載入失敗，會顯示錯誤提示
